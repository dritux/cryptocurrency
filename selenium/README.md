# bitnyx

### System dependencies
```
- python3.7
- python3.7-dev
- python3-pip
- python3-venv
```

# Running in the local environment
```
$ python3 -m venv .venv
$ source .venv/bin/activate
$ make install
```

# Running scripts

```
python bitnyx.py LOGIN PASSWORD
python freebitco.py LOGIN PASSWORD
```
